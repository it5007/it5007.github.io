Welcome to IT5007 Semester 1 AY2022/2023!

In this course, you will learn to design and develop modern applications that are able to support multiple clients across different platforms such as desktop, mobile devices, and cloud.

The course covers:

  1. Manage software development with productivity-enhancing tool chains
  2. Understand web programming concepts
  3. Write web application UI using Javascript, HTML, and CSS
  4. Implement a cloud-based backend for the web application using frameworks such as Express
  5. Understand and implement popular databases such as MongoDB and MySQL in the backend
  6. Implement a mobile app front-end to interact with the backend, e.g. using Express API
  7. Understand the importance of server-side Javascript technologies (e.g., Node.js) and their advantages over traditional server-side frameworks such as Java or Python
  8. Understand and use concepts of DevOps Toolchain
  9. Understand the key interactions between different components of the software stack
  10. Experience building the full-stack of a web and a mobile application
  11. Get a feel of evaluating the higher-level goals of the software such as security and performance, and identify stack-bottlenecks

Tools and techniques for writing modern software, such as, HTML5, CSS3, React.js, Node.js, MySQL/MongoDB, and Git will be taught.
